Forever Web UI beta

Simple web UI for efficient NodeJS administration. Just a little experimentation with Backbone.js

Installation

Via npm (node package manager)

    npm install forever-webui &&
    sudo node node_modules/forever-webui/app.js

and browse http://127.0.0.1:8085

    http://127.0.0.1:8085

Via Git + NPM

    git clone https://github.com/r8k/forever-webui.git
    cd forever-webui/
    npm install
    sudo node app.js

and browse http://127.0.0.1:8085

Usage

Start NodeJS processes with forever cli and manage them via a web interface.

** Please note that you need to start Forever Console via sudo, because if you are planning to run node processes thru sudo, they will be visible to forever, via sudo only. [1]
